# Pathways: Reading Development Equity Simulator

An interactive web-based simulator that models how early working memory, socioeconomic status (SES), and reading support influence reading outcomes.

## Purpose
This project was built to translate educational psychology research into an accessible, interactive experience. It highlights how environmental factors shape learning trajectories and outcomes.

- HTML
- CSS
- JavaScript
- Codex AI
